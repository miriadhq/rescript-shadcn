## Usage

```res showLineNumbers
<Label htmlFor="email"> {"Your email address"->React.string} </Label>
```

## Label in Field

For form fields, use the [Field](/docs/components/base/field) component which
includes built-in `FieldLabel`, `FieldDescription`, and `FieldError` components.

```res showLineNumbers
<Field>
  <Field.Label htmlFor="email"> {"Your email address"->React.string} </FieldLabel>
  <Input id="email" />
</Field>
```
