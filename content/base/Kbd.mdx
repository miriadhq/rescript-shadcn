## API Reference

### Kbd

Use the `Kbd` component to display a keyboard key.

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | ``      |

```res showLineNumbers
<Kbd> {"Ctrl"->React.string} </Kbd>
```

### Kbd.Group

Use the `Kbd.Group` component to group `Kbd` components together.

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | ``      |

```res
<Kbd.Group>
  <Kbd> {"Ctrl"->React.string} </Kbd>
  <Kbd> {"B"->React.string} </Kbd>
</Kbd.Group>
```
