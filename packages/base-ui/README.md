# Base-UI bindings for ReScript

The goal of this package is to provide crude rescript bindings to [base-ui](https://base-ui.com/) components.

To keep things simple, they all share the same permissive set of props. 
This could be improved in future iterations so types are more strict.

These bindings are used in [rescript-shadcn](https://rescript-shadcn.miriad.studio/).
